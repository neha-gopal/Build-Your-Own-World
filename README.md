# Build-Your-Own-World

A tile-based world generation game built in Java where users can create, explore, and interact with randomly generated worlds. The project uses procedural generation to create unique maps each time, giving players a different experience on every run.

Overview  

Build Your Own World is an interactive Java project that generates 2D worlds made up of rooms, hallways, and walls. Users can explore the world through keyboard inputs, and the program supports saving and loading game state so progress can be continued later.   

Features  
-Procedurally generates a new tile-based world from a random seed  
-NPC interaction  
-Audio sound effects  
-Connects rooms with hallways to create a navigable map  
-Player movement through keyboard controls  
-Save and load functionality for restoring previous worlds  
-Deterministic generation so the same seed recreates the same world  
-Structured tile rendering system for visual display
